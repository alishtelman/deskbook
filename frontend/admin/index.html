Шаг 1 — Загрузите план

Выберите этаж из выпадающего списка, затем нажмите «Загрузить план» и загрузите PNG или SVG с планировкой офиса. Он станет фоном для расстановки мест.

Шаг 2 — Расставьте рабочие места

Выберите инструмент «Место» (клавиша D) и кликайте по плану, чтобы разместить рабочие места. Используйте «Зона» (B) для обозначения помещений.

Шаг 3 — Сохраните и опубликуйте

Нажмите «Сохранить» чтобы создать черновик. Когда план готов — «Опубликовать», чтобы сотрудники увидели его в приложении бронирования.
